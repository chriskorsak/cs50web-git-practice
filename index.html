CS50 Web: Git Workflow

  1. Create remote repository on Github
  2. Terminal: navigate to where you want to clone github directory on your local computer, then type: git clone (url of repository)
  3. Create project files on local computer (html, css, js, etc..)
  4. Terminal: git add (name of project file)
  5. Terminal: git commit -m "short message here detailing work you've done on files"
  6. Terminal: git status
    • This tells you what's currently happening in the repository
  7. Terminal: git push

Git tips and shortcuts

  • Instead of typing: git add (name of file), then git commit -m "commit message here", you can just type: git commit -am "message". This adds all files and commits in one command.
